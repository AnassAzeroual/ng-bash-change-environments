## Configuration in you LinuxOS or MacOS

open terminal and run those command
```sh
code ~/.bashrc
```
### Add variables 
```sh
export Angular_v=15
export Ionic_v=6
```
### run in terminal this command
```sh
source ~/.bashrc
```
### To check if the commands are available    
```sh
echo $Angular_v && echo $Ionic_v
```
### To run this script
```sh
npm run config
```
